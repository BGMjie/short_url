# ShortUrl

> 短链接生成应用

## 安装elixir
http://elixir-lang.org/install.html

## 首次运行

  * 安装依赖  `mix deps.get`
  * 创建数据库及数据表  `mix ecto.create && mix ecto.migrate`
  * 安装前端依赖 `cd assets && npm install`
  * 启动服务 `mix phx.server`
  * 访问应用 [`localhost:4000`](http://localhost:4000)
